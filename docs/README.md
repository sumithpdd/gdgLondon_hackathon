## 📦 Project Information

### What This App Does

Build with AI Hackathon platform:

1. **Hackathon Competition**
   - Project submissions (demo video, code repo, screenshots)
   - Project gallery with search, sort, filters
   - Admin panel for winner selection
   - Draft system

2. **AI Devcamp 2026 Mentorship Program**
   - Mentee applications
   - Mentor applications with resume upload
   - Tag-based skill matching
   - Application tracking

### Tech Stack

- **Framework**: Next.js 14.2 (App Router)
- **Language**: TypeScript 5.6
- **Authentication**: Firebase Auth (Email/Password + Google)
- **Database**: Firebase Firestore
- **Storage**: Firebase Storage
- **UI**: shadcn/ui + Tailwind CSS
- **Deployment**: Vercel

### Key Features

- 🔐 Firebase Authentication (no custom domain needed)
- 📝 Draft system for submissions
- 📸 Multi-screenshot upload (up to 5)
- 🏆 Winner selection (1st, 2nd, 3rd place)
- 👥 User role management (Admin, Moderator, User)
- 🎓 Mentorship program with applications
- 🏷️ Smart tag system for skills
- 📄 Resume upload for mentors
- 📱 Fully responsive design
